## About

NextGallery is a free, open-source desktop application for browsing and managing AI-generated images and videos. Built with .NET MAUI and WinUI 3 for native Windows performance.

### Features

- **High Performance** - Virtualized grid handles thousands of images
- **Compare Mode** - Side-by-side comparison with parameter diff
- **Smart Search** - Find by prompt, seed, preset, or date
- **Job Management** - Delete, copy prompt, export metadata
- **ComfyUI Integration** - Works with CodeComfy VS Code extension
- **Video Support** - Preview and organize AI-generated videos

## Download

- Windows 10 version 1809 or later
- .NET 9.0 Runtime (included with installer)
